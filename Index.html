What we do?

Roll forming is a cost effective process for shaping metal coils into designed profiles.
It is a continuous process which converts sheet metal into an engineered shape using consecutive sets of mated rolls,
each of which makes only incremental changes in the form.
We are processing color coated sheets into trapezoidal and sinusoidal profiles along with sheet slitting and bending.

Connect with us for Roll Forming, Slitting and Bending Services to
ensure Uniformality & Similarity

SINUSIODAL PROFILE SHEET


Sinusoidal Profile is a corrugated profile as defined in BS 4868. Featuring a very shallow profile, it is intended for use on relatively steep roof pitches.
